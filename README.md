# Angel Lomelí — Portfolio

Portfolio personal de Angel Lomelí, desarrollador Jr Backend / Fullstack.

Construido con Astro + Tailwind CSS. Bilingüe (EN / ES).

**Live:** https://alomeliBravo.github.io/alomeli-portfolio/

## Stack

- Framework: Astro
- Estilos: Tailwind CSS + CSS custom properties
- i18n: inglés por defecto (`/`), español en `/es/`
- Deploy: GitHub Pages

## Comandos

| Comando          | Acción                                    |
| :--------------- | :---------------------------------------- |
| `npm install`    | Instala dependencias                      |
| `npm run dev`    | Servidor local en `localhost:4321`        |
| `npm run build`  | Build de producción en `./dist/`          |
| `npm run preview`| Preview del build antes de desplegar      |
